# vect2

## Descripción
El main no es el que pasan del examen, pero es más ampliado, por lo que incluye todo lo que pasan en el examen. Lo pongo por que me parece uno de los mejores ejercicios para comprender, para que sirven hacer un objeto. Es decir, componer un nuevo tipo de números.

## Instructiones
1. **Prerequisitos**: Tienes que tener instalado el compilador de C++ en Linux
2. **Compilación**: compilar con:

```bash
c++ -Wall -Wextra -Werror -std=c++98 *.cpp -o vect2
```
2. **Ejecutar**: 
```bash
./vect2
```

Siéntete libre de modificar el main por el del examen o con tus pruebas.
